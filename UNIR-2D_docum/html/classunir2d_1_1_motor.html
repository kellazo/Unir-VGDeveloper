Descripción detallada

Ejecuta el juego definido en una instancia de una clase derivada de JuegoBase.

Al ejecutar el juego aparece una ventana donde se desarrolla el juego y aparece también la consola de Window donde se muestran los mensajes de ejecución del juego.

Ejemplo de uso:

int main () {
try {
// Se crea el juego.
JuegoAnimacion * juego = new JuegoAnimacion {};
// Se crea el motor de juego.
unir2d::Motor * motor = new unir2d::Motor {};
// El motor ejecuta el juego.
motor->ejecuta (juego);
// Se eliminan las instancias.
delete juego;
delete motor;
} catch (std::exception excepcion) {
std::cerr << "UNIR-2D :: Prueba de animación\n";
std::cerr << " " << excepcion.what () << "\n";
std::this_thread::sleep_for (std::chrono::milliseconds (5000));
}
return 0;
}
