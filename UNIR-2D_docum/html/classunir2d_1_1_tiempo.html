Descripción detallada

Realiza mediciones de tiempo.

Esta clase mantiene un contador de tiempo global y un contador de tiempo parcial, denominado cronómetro. También mantiene un contador para la media de las mediciones del cronómetro.

Esta clase es usada internamente por la clase Motor.
