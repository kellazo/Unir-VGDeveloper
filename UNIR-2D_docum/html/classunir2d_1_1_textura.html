Descripción detallada

Almacena una imagen en la memoria.

Para presentar la imagen de una instancia de esta clase en la pantalla se usa una instancia de la clase Imagen o se usa una instancia de la clase Baldosas para cubrir una superficie.

Esta clase se puede usar para cargar las imágenes de un archivo o para crear una imagen nueva en la memoria.

Dibuja sobre la imagen de esta instancia la imagen indicada.
